Photon OS

发表于 2025-12-02

下载地址

安装教程

https://www.itgeeker.net/one-minute-build-esxi-8-photon-ova-vm-to-run-docker-one-vm-run-one-docker/

Photon OS连不上SSH的解决办法

Photon OS默认关闭了Root SSH登陆。
编辑sshd_config

vi /etc/ssh/sshd_config

systemctl restart sshd

没有daemon.json文件，手动创建

新建 /etc/docker/daemon.json文件

重启
systemctl restart docker

测试
docker pull hello-world
