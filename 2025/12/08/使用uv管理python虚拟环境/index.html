使用uv管理python虚拟环境

发表于 2025-12-08

介绍

官网：https://docs.astral.sh/uv/
使用uv来管理虚拟环境，相比pip有至少10倍的性能提升
功能

  • 管理python版本
  • 管理第三方库的版本
  • 拥有全局第三方库的缓存
  • 安装uv不需要python环境
  • 多平台支持

安装uv

linux

windows

mac安装后必须把$HOME/.local/bin这个路径加到环境变量PATH里面才可以正常使用uv工具，设置之后，使用uv –version 确认安装成功

2.1. 卸载uv
卸载uv之前，先执行如下命令删除所有本地缓存文件
uv cache clean
rm -r “$(uv python dir)”
rm -r “$(uv tool dir)”

mac/linux

rm ~/.local/bin/uv ~/.local/bin/uvx

windows

rm $HOME.local\bin\uv.exe
rm $HOME.local\bin\uvx.exe
